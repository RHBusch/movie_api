index.js

the Index.js file is the nexus of all major aspects of the API. It establishes the Express app and creates the server. API endpoints are created for all app requests. Request routes follow the mongoose models detailed in models.js. Authentication is managed in the passport.js file. The database is powered by MongoDB Atlas. The server is powered by Heroku.
