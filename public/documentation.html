Documentation: myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request HTTP Method URL Request Body Data Format Response Body Data Format
Return all movies to user GET '/movies' N/A url parameter doesn't require example. A JSON object is returned to the user including all movies in an array.
Return data about a single movie GET '/movies/:Title' N/A url parameter doesn't require example. A JSON object is returned to the user based on the title parameter included in the url.
Return movies within a specific genre. GET '/movies/genre/:Name' N/A url parameter doesn't require example. A JSON object is returned including all movies within a specified genre based on the parameter in the url.
Return movies from a specific director by name. GET '/movies/director/:Name' N/A url parameter doesn't require example. A JSON object is returned including all movies with a specific director based on the parameter in the url.
Allow new users to register. POST '/users' A JSON object w/the data of a new user.
Example:
{ "username": "TLasso21"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"Birthday": "1993-12-25"
} 		Creating a JSON object holding data about the newly registered user.
Example:
{ "_id":"9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d" "username": "TLasso21"
"Email": "TLasso21@gmail.com"
"password": "HateNate21"
"Birthday": "1993-12-25"
"favoriteMovies": []
}
Allow users to update their user info. PUT '/users/:Username' A JSON object w/ desired updated data and username as a confirmation from user.
Example:
{ "username": "TLasso21"
"Email": "TLasso21@gmail.com"
"password": "Kansas2London"
"Birthday": "1993-12-25"
"favoriteMovies": []
} 		Updated JSON object holding new data about an existing user.
Example:
{ "_id":"9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "Kansas2London"
"favoriteMovies": []
}
Allow users to add a movie to their list of favorites. POST '/users/:Username/movies/:MovieId' Adding a movieId to the url to add a new favorite movie. For example: http://localhost:8080/users/RossTest/movies/61a36ee44c95362c52701d78 Creating a JSON object including user info and the the added movie.
Example:
{ "_id": "61a6d627e8b060b3bd9abaee", "Username": "RossTest", "Password": "RossTestPW", "Email": "RossNewNew@gmail.com", "FavoriteMovies": [ "61a36ee44c95362c52701d78" ], "__v": 0, "Birthday": "1973-12-30T00:00:00.000Z" }
Allow users to remove a movie from their list of favorites. DELETE '/users/:Username/remvmovies/:MovieId' Adding a movieId to the url to remove a favorite movie. For example: http://localhost:8080/users/RossTest/remvmovies/61a36ee44c95362c52701d78 Returning a JSON object with user info and without the deleted movie in the parameter.
Example:
{ "_id": "61a6d627e8b060b3bd9abaee", "Username": "RossTest", "Password": "RossTestPW", "Email": "RossNewNew@gmail.com", "FavoriteMovies": [ ], "__v": 0, "Birthday": "1973-12-30T00:00:00.000Z" }
Allow users to deregister. DELETE '/users/remv/:Username' N/A Removing a JSON object holding data about an existing user and returning confirmation text.
Example:

"(USERNAME) was deleted"
