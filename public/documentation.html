Documentation: myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update theirpersonal information, and create a list of their favorite movies.

Request HTTP Method URL Request Body Data Format Response Body Data Format
Return all movies to user GET '/movies' N/A A JSON object is returned to the user including all movies in an array.
Return data about a single movie GET '/movies/:title' N/A A JSON object is returned to the user based on the title paramter included in the url. For example:
Return movies within a specific genre. GET '/movies/:genre' N/A A JSON object is returned including all movies within a specified genre based on the parameter in the url.
Return data about a specific director by name. GET '/movies/:director' N/A A JSON object is returned including all Movies with a specific director based on the parameter in the url.
Allow new users to register. POST '/users' ### Creating a JSON object holding data about the newly registered user.
Allow users to update their username. PUT '/users/:username' ### Updating a JSON object holding data about an existing user.
Allow users to add a movie to their list of favorites. POST '/users/add/:movieTitle' ### Creating a JSON object based on the added movie and returning confirmation text.
Allow users to remove a movie from their list of favorites. DELETE '/users/remove/:movieTitle' N/A Removing a JSON object based on the deleted movie in the parameter and returning confirmation text.
Allow users to deregister. DELETE '/users/remove/:username' N/A Removing a JSON object holding data about an existing user and returning confirmation text.
