Documentation: myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update theirpersonal information, and create a list of their favorite movies.

Request HTTP Method URL Request Body Data Format Response Body Data Format
Return all movies to user GET '/movies' N/A url parameter doesn't require example. A JSON object is returned to the user including all movies in an array.
Return data about a single movie GET '/movies/:title' N/A Url parameter doesn't require example. A JSON object is returned to the user based on the title parameter included in the url.
Return movies within a specific genre. GET '/movies/:genre' N/A Url parameter doesn't require example. A JSON object is returned including all movies within a specified genre based on the parameter in the url.
Return data about a specific director by name. GET '/movies/:director' N/A Url parameter doesn't require example. A JSON object is returned including all movies with a specific director based on the parameter in the url.
Allow new users to register. POST '/users' A JSON object w/the data of a new user.
Example:
{ "firstName": "Ted"
"lastName": "Lasso"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "HateNate21"
} 		Creating a JSON object holding data about the newly registered user.
Example:
{ "_id":"9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d" "firstName": "Ted"
"lastName": "Lasso"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "HateNate21"
"favoriteMovies": []
}
Allow users to update their user info. PUT '/users/:username' Creating a JSON object w/ desired updated data and username as a confirmation from user.
Example:
{ "username": "TLasso21"
"password": "Kansas2London"
} 		Updated JSON object holding new data about an existing user.
Example:
{ "_id":"9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d"
"firstName": "Ted"
"lastName": "Lasso"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "Kansas2London"
"favoriteMovies": []
}
Allow users to add a movie to their list of favorites. POST '/users/add/:movieTitle' JSON Object w/user information including new movies to add to their favorites.
Example:
{ "firstName": "Ted"
"lastName": "Lasso"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "Kansas2London"
"favoriteMovies": ["Random Movie"]
} 		Creating a JSON object based on the added movie and returning confirmation text.
Example:

"Another favorite movie was successfully added to your list."
Allow users to remove a movie from their list of favorites. DELETE '/users/remove/:movieTitle'
Example:
{ "_id":"9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d"
"firstName": "Ted"
"lastName": "Lasso"
"Email": "TLasso21@gmail.com"
"username": "TLasso21"
"password": "Kansas2London"
"favoriteMovies": []
} 		Removing a JSON object based on the deleted movie in the parameter and returning confirmation text.
Example:

"A movie was successfully deleted from your list."
Allow users to deregister. DELETE '/users/remove/:username' N/A Removing a JSON object holding data about an existing user and returning confirmation text.
Example:

"You have successfully deregistered."
