Documentation: myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update theirpersonal information, and create a list of their favorite movies.

Action Request URL Response
Return all movies to user GET '/movies' All movies in database are returned as an array.
Return data about a single movie GET '/movies/:title' A single movie and its data is returned to a user based on its title.
Return movies within a specific genre. GET '/movies/:genre' All movies within a specific genre are returned to a user.
Return data about a specific director by name. GET '/movies/:director' Movies with a specific director are returned to a user.
Allow new users to register. POST '/users' ...
Allow users to update their username. PUT '/users/:username' ...
Allow users to add a movie to their list of favorites. POST '/users/add/:movieTitle' ...
Allow users to remove a movie to their list of favorites. DELETE '/users/remove/:movieTitle' ...
Allow users to deregister. DELETE '/users/remove/:username' ...
